URL Validation Rules

A valid URL must:

  • Start with http:// or https://
  • Contain a valid domain name (e.g., example.com)
  • Optionally include a path or subdirectory (e.g., /path)
  • Can include query parameters (e.g., ?id=123&ref=promo)
  • Cannot contain spaces
  • Use valid characters like letters, numbers, hyphens, and dots
